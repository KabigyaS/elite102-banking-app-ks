# Banking App Project
This is a simple command-line banking application built using Python and SQL. It allows users to create accounts, deposit and withdraw money, check balances, and view transaction history, all stored in a local database.

Features:
Create a new bank account, Login to an existing account, Deposit money, Withdraw money (with balance check), View current balance, View transaction history, List all accounts

Environment Requirements:
Python 3.x
No external libraries required (uses built-in sqlite3)
